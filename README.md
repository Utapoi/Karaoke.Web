## ðŸ”§Getting Started

<br>

### Prerequisites

We use pnpm for package management.

```sh
npm i pnpm@lastest -g
```

<br>

### Installation

Clone the repository and install the required packages.

```sh
pnpm i
```

<br>

### Usage

To run the project in local, you just need to use the following command.

```sh
pnpm run dev
```
